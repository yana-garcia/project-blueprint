How to use?


Create a section by drag and dropping the 'Section' component into the stage and within the section, drag and drop the 'HTML' component.

Upon selecting which component to use, you can simply copy the HTML code provided in the repository site buy clicking the 'Copy Code' button.

A downloadable template is also available for use for wireframing by clicking the 'Downloadable XD File'.

Paste the code on the HTML section to replicate the desired component into the Bundles site.

Look for the line '<!-- slide 1 -->'
to edit the carousel's main slide.

Look for 'style= "background:'
to to edit the slide's color.

To edit the slide's icon, look for the line 'img src=' and highlight the image link and click 'Browse Assets' to upload an image to replace the placeholder.

And for the slide's title, look for the line '<h1>' and edit the text within the line.

Look for the line '<!-- slide 1-1 -->'
to edit the carousel's sub-slide (slide below the main).

Look for 'style= "background:'
to to edit the slide's color.

And for the slide's text content, look for the line '<h1>' and edit the text within the line.

NOTE:
Lines such as '<!-- slide 1 -->' and '<!-- slide 2 -->' indicates main slides (horizontal slides) of the carousel.

While lines such as '<!-- slide 1-1 -->'   '<!-- slide 1-2 -->' indicates sub-slides (vertical slides) of the carousel.
